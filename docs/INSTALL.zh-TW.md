# palserver GUI 安裝與連線指南(玩家版)

這份是給「想開一台 Palworld 伺服器」的玩家看的。不需要懂程式,照著做就好。

---

## 一、下載

到 [Releases 頁面](https://github.com/io-software-ai/palserver-gui/releases) 下載對應你系統的檔案:

| 你的系統 | 下載 | 內容 |
| --- | --- | --- |
| Windows | `palserver-agent-windows.zip` | 免安裝執行檔 + 網頁介面(最常用) |
| Linux | `palserver-agent-linux.zip` | 同上 |
| macOS | `palserver-agent-macos.zip` | 同上(Mac 無法實際跑 Palworld 伺服器,僅供管理遠端主機) |

解壓縮後會看到:

```
palserver-agent(.exe)   ← 主程式,雙擊執行
web/                    ← 網頁介面(別刪,程式會自動用它)
```

> **不用另外安裝 Node.js 或任何東西**,執行檔已經把需要的都包進去了。

---

## 二、啟動

雙擊 `palserver-agent.exe`(Windows)或在終端機執行 `./palserver-agent`(Mac/Linux)。

- Windows 第一次執行,SmartScreen 可能跳「不明發行者」警告 → 按「其他資訊」→「仍要執行」。
- 視窗會保持開著,關掉視窗 = 關掉伺服器管理服務,所以先讓它開著。

啟動後視窗會顯示類似這樣的訊息:

```
  ┌───────────────────────────────────────────────
  │  palserver GUI agent 已啟動
  │
  │  本機管理(免密碼,直接打開):
  │      http://localhost:8250
  │
  │  同一區網 / VPN 的裝置可連:
  │      http://100.86.188.111:8250   (Tailscale)
  │      http://192.168.0.19:8250
  │
  │  邀請朋友遠端連線 —— 把這條連結傳給他:
  │      http://100.86.188.111:8250/?setup=EEHN-CVMY
  │  或請對方在網頁輸入配對碼:EEHN-CVMY
  └───────────────────────────────────────────────
```

---

## 三、開始管理(在自己這台電腦)

直接用瀏覽器打開 **http://localhost:8250** —— 會直接進入管理介面,**不需要輸入任何密碼**(本機連線自動信任)。

接著就能建立伺服器、改設定、備份、看線上玩家等。

---

## 四、讓朋友一起玩

Palworld 專用伺服器要能被朋友連到,有兩條路:

### 路線 A:VPN(推薦,最簡單也最安全)

你和朋友裝同一套免費 VPN、加入同一個網路,就像在同一個 WiFi 裡,不用動路由器設定。

- **Radmin VPN** — 免註冊,建個房間邀朋友加入,最適合遊戲聯機。
- **Tailscale** — 用 Google/GitHub 帳號登入,穩定,適合長期。

裝好 VPN 後,把 agent 視窗上的 **VPN 位址**(通常是 `100.x.x.x`)給朋友。他要進「管理介面」的話,你再把 `?setup=配對碼` 那條連結給他即可。

> **VPN 本身就是加密通道**,所以就算走的是 `http`,在 VPN 內也是安全的。

### 路線 B:公開 IP 直連(進階)

需要處理路由器連接埠轉發、防火牆、浮動 IP / CGNAT 等,新手較麻煩。我們有提供「IP 直連設定服務」協助設定,詳見介面內說明。

## 五、關於管理介面的連線與安全

| 情境 | 會怎樣 |
| --- | --- |
| 自己這台電腦開 `localhost` | 直接進,免密碼 |
| 同區網/VPN 的另一台裝置 | 需要**配對碼**(agent 視窗上有),或點 `?setup=` 連結一次即可記住 |
| 公開網站版(未來)連自己家的 agent | 受瀏覽器限制,建議走 VPN 或合一版本機管理 |

進階選項(環境變數):

- `PALSERVER_REQUIRE_TOKEN=1` — 連本機也強制要配對碼(多人共用的電腦建議開)。
- `PALSERVER_TLS=1` — 用 HTTPS 監聽(自簽憑證會跳瀏覽器警告;想要公開直連的加密才需要)。
- `PALSERVER_AGENT_PORT=8250` — 改連接埠。
- `PALSERVER_WEB_ORIGINS=https://你的公開站` — 允許某個公開網站跨來源連這台 agent。

## 六、常見問題

**朋友連不進遊戲伺服器?**
先確認伺服器正在運作中、防火牆有放行「遊戲埠(UDP)」,以及朋友用的是正確的位址。

**打開網頁看到的是舊畫面?**
更新 agent 後按一次 `Ctrl+Shift+R`(強制重新整理)。

**配對碼在哪?**
在 agent 啟動的視窗裡。忘了可以在管理介面重新產生。

**要換一台電腦管理?**
把 agent 的 VPN 位址 + 配對碼給那台裝置即可,存檔和設定都在跑 agent 的主機上。
