為什麼需要它

開專用伺服器, 不該是一場惡夢。

改不完的 .ini 設定檔、背不完的 RCON 指令、朋友卡在「連不進來」、遊戲一改版就炸存檔、半夜伺服器崩潰沒人救……這些不是「架好就沒事」的小事。palserver GUI 把每一項都收進乾淨的畫面裡, 新手覺得簡單, 老手覺得夠力。

一分鐘看懂

Agent + 網頁, 人在哪都能顧。

伺服器主機上跑一支輕量的 agent 常駐程式; 你在任何裝置打開網頁介面連上去。存檔與運算都在主機, 網頁只是遙控器——本機免密碼, 遠端用好念的配對碼或一條設定連結。

你的裝置
電腦、手機、平板。打開網頁就是管理畫面, 本機直進、遠端點連結即連。

agent 常駐, 直接開 PalServer(原生)或跑 Docker 容器, 管理存檔、模組與備份。
功能巡禮

從開服到救火, 一頁全包。

每一台伺服器點進去就是完整面板:總覽、效能、玩家、地圖、指令、設定、引擎、模組、備份、自動重啟、日誌——12 個分頁一行排開。

Dashboard

所有伺服器, 一眼掌握

每台的狀態、原味/強化、遊戲埠、可更新提示都在卡片上, 點進去就是完整管理。

Settings & Tuning

設定與引擎微調, 附白話說明

世界規則、經驗倍率、掉落、PvP 全部有中文標籤與提示; 進階的 Engine.ini Tick 率、頻寬、GC 也備好預設組合。

  • 每一項都告訴你調高調低會怎樣
  • 寫檔保留你手動加的鍵
Mods

PalDefender / UE4SS 一鍵管理

反外掛與模組載入器一鍵裝、更新、移除, 還會提醒「遊戲改版後模組可能暫時失效」。Lua 模組直接在畫面上開關。

Performance

即時效能, 走勢一目了然

CPU、記憶體、行程數、運行時間, 加上伺服器 FPS 與影格時間, 配上即時走勢圖。撐不住的時候, 你會第一個知道。

World settings

80+ 世界參數, 不用開檔案

難度、資源、繁殖、據點、傷害倍率……全部整理成分類、附說明的表單, 改完提示你重啟生效。存檔損壞還會偵測並一鍵重建。

Multi-device

換裝置、邀朋友, 一頁搞定

設定頁給你配對碼與各個可連位址組好的「一鍵登入連結」, 複製給手機或另一台電腦就能連; 也能一鍵清除瀏覽器暫存重連。

兩種人都合用

新手覺得簡單, 老手覺得夠力。

For beginners

第一次開伺服器

  • 一鍵開服。 建立、啟動、更新全部用按的, 不背指令、不改設定檔。
  • 本機免密碼直進。 在自己電腦打開就是管理畫面, 零設定。
  • 邀朋友零門檻。 傳一條設定連結或配對碼, 朋友點一下就連上。
  • 可愛又直覺。 中文介面、附說明, 滑鼠點一點就能調。
For power users

老手想要的控制力

  • 原生 / Docker 雙後端。 直接開 PalServer 或跑容器; 可接管既有安裝或指定空資料夾安裝。
  • Schema 驅動設定。 80+ 世界參數 + Engine.ini 引擎微調, 型別一致、保留未管理的鍵。
  • RCON 指令台 + 模組。 內建 RCON; PalDefender / UE4SS 一鍵裝更新移除。
  • 備份排程與遷移。 tar.gz 定期備份、一鍵還原、跨來源存檔搬家、REST API 代理。
社群一路陪著長大

你許願的, 我們都做了。

palserver GUI 從一個「懶得改設定檔」的小工具開始, 這兩年社群提的需求, 一個一個補上。

「能不能把既有的存檔導進去?」
存檔遷移——別台專用伺服器、v1 舊版、本機四人邀請碼存檔都能接管。
「好怕存檔壞掉…」
備份排程 + 一鍵還原, 還會偵測存檔損壞並協助重建。
「想開伺服器玩模組」
PalDefender / UE4SS 一鍵安裝、更新、移除,Lua 模組畫面上開關。
「死亡掉落、孵化時間也能調嗎?」
80+ 世界參數視覺化, 分類、附說明, 不用再開 .ini。
「朋友延遲太高連不進來」
VPN 一鍵邀請, 或選公司的 IP 直連設定服務。
「不想每次都打一長串指令更新」
版本檢查 + 一鍵更新, 零指令。
三步開始

下載、執行、打開瀏覽器。

不用先裝 Node、不用 Docker、不用命令列。免安裝執行檔把需要的都包好了。

下載執行檔

到 Releases 下載你系統的 palserver-agent, 解壓縮。

雙擊執行

視窗會印出本機網址與邀朋友的配對連結, 讓它開著就好。

打開瀏覽器

連 localhost:8250 直接進管理畫面, 開你的第一台伺服器。

細節控的貼心

連「開場白」都幫你想好了。

內建公告系統、存檔損壞偵測與一鍵重建、自動重啟(排程 / 記憶體門檻 / 崩潰救援)、Steam ID 全站打碼、線上地圖、三語介面與深淺色——很多你之後才會感謝的小地方。
